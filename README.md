# Missão 42 - Imersão Python

Aqui temos o que foi trabalhado durante a Missão 42 na 42 São Paulo. O objetivo foi uma imersão prática na sintaxe e nas estruturas fundamentais do Python, focando na lógica de programação e manipulação de dados utilizando a metodologia de aprendizagem Peer-To-Peer.

## Conceitos Abordados

Os exercícios cobrem desde a sintaxe básica até algoritmos de ordenação manual, incluindo:

* Interação com Sistema (CLI): Manipulação intensiva de argumentos de linha de comando com `sys.argv`.
* Controle de Fluxo: Estruturas condicionais (`if/else`) e loops (`while`) para iteração e validação lógica.
* Manipulação de Strings: Fatiamento (slicing), formatação (f-strings), métodos de texto (`lower`, `capitalize`, `find`) e Regex básico (`re`).
* Estruturas de Dados:
    * Listas: Criação, modificação, filtragem e iteração.
    * Dicionários: Acesso a chaves/valores, dicionários aninhados e iteração complexa.
    * Sets: Remoção de duplicatas.
* Funções: Definição, parâmetros (default e posicionais), retorno de valores e funções aninhadas.
* Algoritmos:
    * Cálculo de médias e operações matemáticas.
    * Lógica de Bubble Sort para ordenação manual de listas e dicionários (sem uso de `lambda` ou `sort` nativo).
    * Filtros condicionais.

## Tecnologias
* Python 3
* Bibliotecas padrão: `sys`, `re`
